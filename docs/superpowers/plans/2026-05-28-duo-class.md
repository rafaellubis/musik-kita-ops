# DUO Class Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Menambahkan tipe kelas DUO (dua murid belajar bersama satu guru) ke sistem Musik KITA — termasuk paket, honor, conflict detection, dan UI.

**Architecture:** `class_type = 'DUO'` baru di tabel `packages`. Dua murid DUO masing-masing punya enrollment + schedule + session sendiri di DB. Conflict detection di-bypass untuk pasangan DUO (seperti pola KIDS_CLASS). Honor guru per murid dikonfigurasi via `PayrollConfig` dengan scenario_code `H_DUO`.

**Tech Stack:** Laravel 11, MySQL, Blade + Tailwind CSS, Spatie Permission, AlpineJS.
