# Bug Log — Musik KITA Ops

Daftar semua bug yang ditemukan selama development, beserta status dan penjelasannya.
Diisi setiap kali ditemukan bug — baik saat testing manual, testing otomatis, maupun laporan dari pengguna.

---

## Format Penamaan File

```
BUG-NNN-slug-deskripsi-singkat.md
```

Contoh: `BUG-001-required_if-operator-tidak-valid.md`

## Status Definisi

- **FOUND** — Bug ditemukan, belum dianalisis
- **CONFIRMED** — Bug dikonfirmasi bisa direproduksi
- **FIXED** — Bug sudah diperbaiki dan di-commit
- **WONTFIX** — Diketahui tapi tidak akan diperbaiki (dengan alasan)
